Project Overview
La Trattoria is a fully responsive static, single-page landing website for a modern Italian restaurant. The site is built with HTML5, CSS3, and vanilla JavaScript and requires no backend and no setup.

Main entry: index.html

Website Objectives
Attract customers with an elegant, mobile-friendly online presence.

Showcase authentic Italian cuisine with clear pricing and high-quality images.

Provide smooth navigation and engaging user interactions (hamburger menu + menu tabs).

Offer an easy way to make reservations via the booking form.

Connect customers with the restaurant through social media integration.
